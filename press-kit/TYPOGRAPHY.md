# Q8JAN Typography

Version 1.0

Last Updated: July 2026

---

# Introduction

Typography plays an important role in maintaining a consistent and professional identity for the Q8JAN project.

These guidelines define the preferred fonts and text styles for official use.

---

# Primary Font

Font Family

Arial

Usage

- Website body text
- Documentation
- User interface
- General communication

---

# Alternative Fonts

When Arial is unavailable, the following fonts may be used:

- Helvetica
- Segoe UI
- Roboto
- Sans-serif

---

# Headings

Headings should be:

- Bold
- Clear
- Easy to read
- Consistent across all documents

Recommended hierarchy:

H1 – Main Titles

H2 – Section Titles

H3 – Subsections

---

# Body Text

Body text should be:

- Professional
- Easy to read
- Left aligned
- Consistently spaced

Recommended line spacing:

1.5

---

# Font Sizes

Main Title (H1)

36–48 px

Section Title (H2)

24–32 px

Subheading (H3)

18–22 px

Body Text

16 px

Small Notes

12–14 px

---

# Emphasis

Use **bold** only for important information.

Avoid excessive use of:

- ALL CAPITAL LETTERS
- Multiple font families
- Decorative fonts
- Excessive italic formatting

---

# Consistency

Always use the same typography across:

- Official website
- Documentation
- Whitepaper
- Presentations
- Press releases
- Marketing materials
- Social media graphics

---

# Accessibility

Typography should remain readable on both desktop and mobile devices.

Maintain sufficient contrast between text and background.

Avoid overly small font sizes.
