# About Q8JAN

Q8JAN is a fixed-supply TRC-20 token built on the TRON blockchain.

The project is designed around simplicity, transparency, and long-term sustainability.

Key features include:

- Fixed supply of 100,000,000,000 Q8JAN
- No mint function
- Burn supported
- Zero transaction tax
- Open-source smart contract
- Public documentation
- Community-focused development
