# Q8JAN Logo Usage

Version 1.0

Last Updated: July 2026

---

# Introduction

This document defines the official rules for using the Q8JAN logo across digital and printed media.

Following these guidelines helps maintain a consistent and professional visual identity.

---

# Official Logo Files

The official logo files are located in:

press-kit/logos/

Available formats:

- logo.png
- logo.svg
- favicon.ico

Always use these official files.

---

# Approved Usage

The Q8JAN logo may be used for:

- Official website
- GitHub repository
- Exchange listings
- Wallet listings
- Press articles
- News publications
- Community posts
- Educational content
- Marketing materials

---

# Minimum Size

Digital:

Minimum width: 32 pixels

Print:

Minimum width: 15 mm

The logo should always remain clear and readable.

---

# Clear Space

Always leave adequate empty space around the logo.

No text, graphics, or other elements should overlap or touch the logo.

A recommended minimum clear space is equal to the height of the "Q" in the logo.

---

# Backgrounds

Preferred backgrounds:

- White
- Black
- Dark Blue
- Transparent

Ensure sufficient contrast between the logo and the background.

---

# Do Not

Never:

- Stretch the logo
- Compress the logo
- Rotate the logo
- Change its colors
- Apply gradients
- Add shadows
- Add outlines
- Crop the logo
- Animate the logo without approval
- Replace any design element

---

# File Selection

Use:

PNG
For websites, presentations, and documents.

SVG
For scalable graphics, print, and high-resolution displays.

ICO
For browser favicon and application icons.
