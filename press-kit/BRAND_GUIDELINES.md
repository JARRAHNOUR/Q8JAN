# Introduction

The Q8JAN brand represents transparency, simplicity, security, and long-term sustainability.

These guidelines ensure that the Q8JAN identity remains consistent across websites, exchanges, media articles, presentations, and social platforms.

# Brand Values

The Q8JAN brand is built upon the following principles:

- Transparency
- Simplicity
- Security
- Community
- Long-Term Sustainability
- Professionalism

---

# Official Logo

Always use the official logo included in this Press Kit.

Available formats:

- PNG
- SVG
- ICO

Do not recreate the logo manually.

---

# Logo Usage

Always maintain sufficient clear space around the logo.

The logo must remain clearly visible on all backgrounds.

Preferred logo versions:

- Full Color
- Dark Background Version
- Light Background Version

---

# Incorrect Usage

The following modifications are not permitted:

- Stretching
- Squashing
- Rotating
- Changing colors
- Adding effects
- Cropping
- Replacing typography
- Using unofficial versions

---

# Brand Voice

The Q8JAN communication style should be:

- Professional
- Clear
- Honest
- Transparent
- Confident
- Simple

Avoid exaggerated marketing claims or unrealistic promises.

---

# Visual Consistency

When creating marketing materials, maintain consistent use of:

- Official logo
- Official colors
- Official typography
- Consistent spacing
- High-quality graphics
