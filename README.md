# Overview

Q8JAN is a fixed-supply blockchain token designed with simplicity, transparency, and long-term sustainability.

The project follows a clean tokenomics model:

- Fixed Supply
- No Mint
- No Transaction Tax
- Burn Supported
- Open Source

# TRON Development

Move into the TRON workspace:

```bash
cd tron
```

Install dependencies:

```bash

```

Compile:

```bash
npm run compile
```

Deploy:

