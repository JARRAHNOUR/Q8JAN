# Introduction

Q8JAN est un actif numérique TRC-20 à offre fixe, construit sur la blockchain TRON.

Le projet adopte une philosophie simple et transparente, axée sur la durabilité à long terme.

Les principes fondamentaux comprennent :

- Transparence
- Offre fixe
- Aucune émission cachée
- Aucune taxe sur les transactions
- Projet porté par la communauté

# Fonctionnalités principales

## Offre fixe

L’intégralité de l’offre est créée une seule fois lors du déploiement.

Aucun jeton supplémentaire ne pourra être créé.

---

## Prise en charge de la destruction

Les détenteurs peuvent détruire définitivement leurs propres jetons.

---

## Émission désactivée définitivement

Aucun jeton supplémentaire ne peut être créé après le déploiement.

---

## Aucune taxe sur les transactions

Q8JAN n’impose aucune taxe sur les transactions ni aucuns frais de transfert cachés.

---

## Contrat intelligent transparent

Le contrat intelligent est open source et ne contient aucune fonction cachée permettant de créer de nouveaux jetons.
