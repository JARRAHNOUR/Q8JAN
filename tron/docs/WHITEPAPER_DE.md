# Einleitung

Q8JAN ist ein TRC-20-Digitalwert mit festem Angebot, der auf der TRON-Blockchain basiert.

Das Projekt verfolgt eine einfache und transparente Philosophie mit Fokus auf langfristige Nachhaltigkeit.

Die Grundprinzipien sind:

- Transparenz
- Festes Angebot
- Keine versteckte Token-Erzeugung
- Keine Transaktionssteuer
- Community-orientiert

# Hauptmerkmale

## Festes Angebot

Das gesamte Token-Angebot wird einmalig bei der Bereitstellung des Smart Contracts erstellt.

Es können niemals zusätzliche Token erzeugt werden.

---

## Burn-Funktion

Token-Inhaber können ihre eigenen Token dauerhaft vernichten.

---

## Minting dauerhaft deaktiviert

Nach der Bereitstellung können keine neuen Token mehr erstellt werden.

---

## Keine Transaktionssteuer

Q8JAN erhebt keine Transaktionssteuer und keine versteckten Übertragungsgebühren.

---

## Transparenter Smart Contract

Der Smart Contract ist Open Source und enthält keine versteckten Funktionen zur Token-Erzeugung.
