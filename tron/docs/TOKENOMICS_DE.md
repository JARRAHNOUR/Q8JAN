## Angebotsmodell

Q8JAN verfügt über ein dauerhaft festes Token-Angebot.

Das gesamte Angebot wird einmalig bei der Bereitstellung des Smart Contracts erzeugt.

Es können niemals zusätzliche Token erstellt werden.
