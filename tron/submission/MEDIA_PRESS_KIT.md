# Project Summary

Q8JAN is a fixed-supply TRC-20 token built on the TRON blockchain.

The project is designed around transparency, simplicity, and long-term sustainability. Q8JAN features a permanently fixed token supply, zero transaction tax, and openly documented project resources.

# Token Information

**Name:** Q8JAN

**Symbol:** Q8JAN

**Network:** TRON

**Standard:** TRC-20

**Total Supply:** 100,000,000,000 Q8JAN

**Decimals:** 18

**Transaction Tax:** 0%

**Minting:** Disabled Forever

**Burn:** Supported
