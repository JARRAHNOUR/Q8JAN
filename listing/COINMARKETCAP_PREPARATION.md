## Project Description

Q8JAN is a decentralized TRC-20 token built on the TRON blockchain with a permanently fixed supply of 100,000,000,000 tokens. The project is designed around transparency, simplicity, and long-term sustainability. The smart contract permanently disables minting, supports token burning, and applies zero transaction tax.
