## Project Overview

Q8JAN is a fixed-supply TRC-20 token built on the TRON blockchain. The project is designed with simplicity, transparency, and long-term sustainability in mind. The smart contract has a permanently fixed supply, does not allow additional minting, supports token burning, and applies zero transaction tax.
