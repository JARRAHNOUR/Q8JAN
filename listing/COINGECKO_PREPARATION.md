## Full Description

Q8JAN is a decentralized digital asset built on the TRON blockchain using the TRC-20 standard. The project features a permanently fixed supply of 100,000,000,000 tokens with no mint function, support for token burning, and zero transaction tax. Q8JAN emphasizes transparency, open-source development, and long-term community growth through clear documentation and a straightforward token economy.
