# Accessibility

Maintain a minimum contrast ratio of 4.5:1 for all text.

Avoid using Gold text on White backgrounds.

Prefer White text on Dark backgrounds.

---

# Usage Rules

Gold should be used only for important elements.

Dark backgrounds are preferred.

Avoid using more than three primary colors on the same page.

Maintain visual consistency across all official platforms.
