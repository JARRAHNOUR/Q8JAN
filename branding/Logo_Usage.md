# Q8JAN Logo Usage Guide

Version: 1.0.0
Status: Official

---

# Purpose

This document defines the official usage rules for the Q8JAN logo.

Following these guidelines ensures a consistent and professional visual identity across all platforms.

---

# Official Logo

The official logo is located in:

logo/logo.png

Only this logo should be used for official communications.

---

# Minimum Size

Digital

Minimum Width:
32 px

Website Header

Recommended Width:
48–64 px

Documents

Minimum Width:
25 mm

---

# Safe Area

Always leave clear space around the logo.

Minimum spacing:

50% of the logo height on all sides.

No text or graphics should enter this area.

---

# Approved Backgrounds

✅ White

✅ Black

✅ Dark Navy

✅ Transparent

---

# Unapproved Backgrounds

❌ Busy patterns

❌ Low-contrast colors

❌ Images that reduce readability

---

# Do Not

Never:

• Stretch the logo

• Rotate the logo

• Change its colors

• Add shadows

• Add outlines

• Add effects

• Change proportions

• Remove elements

---
